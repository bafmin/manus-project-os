# PROJECT_CHARTER

This document defines the purpose, scope, and boundaries of the software system being built. It establishes the foundation Manus should use when planning and implementing work.

Every project using the Manus Project OS must complete this document before development begins.

## Definition of Project Start

A project may begin development once the following are complete:

- This charter is filled out
- Architecture overview exists
- Engineering standards are defined
- Test policy is defined
- Release process is defined

Only after these conditions are met should implementation work begin.
