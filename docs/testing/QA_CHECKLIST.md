# QA Checklist

## Scope

- Feature tested against acceptance criteria
- Related workflows checked
- Role differences checked

## Functional

- Create flow works
- Update flow works
- Delete or archive flow works if applicable
- Validation errors display correctly
- Empty and error states behave correctly

## Permissions

- Correct role can access feature
- Incorrect role is blocked
- Ownership rules enforced

## UI

- Layout stable
- No broken buttons or links
- No obvious copy errors
- Loading states acceptable

## Data

- Data saves correctly
- Data reloads correctly
- Audit entries created if required

## Release Confidence

- No blocking defects
- Known issues logged
- Recommendation: ship or hold
